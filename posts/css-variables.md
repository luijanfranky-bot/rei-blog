# CSS 变量的妙用

CSS 自定义属性（CSS Variables）是现代 CSS 中最强大的特性之一。它让我们能够创建更加灵活、可维护的样式系统。

## 什么是 CSS 变量？

CSS 变量，也称为自定义属性，允许我们在 CSS 中定义可重用的值。通过使用 `--` 前缀定义变量，我们可以在整个样式表中引用这些值。

## 基本用法

定义和使用 CSS 变量非常简单：

```css
:root {
  --primary-color: #d97757;
  --spacing-unit: 1rem;
}

.button {
  background-color: var(--primary-color);
  padding: var(--spacing-unit);
}
```

## 实际应用场景

### 1. 主题切换

使用 CSS 变量可以轻松实现明暗主题切换，只需要改变根元素的变量值即可。

### 2. 响应式设计

结合媒体查询，CSS 变量可以帮助我们创建更加灵活的响应式布局。

### 3. 组件样式管理

在组件级别定义变量，可以让组件更加独立和可复用。

## 总结

CSS 变量为我们提供了强大的工具来创建更加灵活和可维护的样式系统。通过合理使用 CSS 变量，我们可以：

- 减少代码重复
- 提高样式的可维护性
- 轻松实现主题切换
- 创建更加灵活的响应式设计
